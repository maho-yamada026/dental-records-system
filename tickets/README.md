# 歯科カルテシステム 開発チケット一覧

矯正歯科患者向け診療履歴管理Webシステムの開発チケット。

## 技術スタック
- **フロントエンド**: Next.js (App Router) + TypeScript + Tailwind CSS
- **バックエンド**: Next.js API Routes
- **DB**: PostgreSQL + Prisma ORM
- **認証**: next-auth v5

## 注意事項
- **サンプルデータ**: `fixtures/` ディレクトリに開発用CSVとフォーマット定義を用意済み。TICKET-003はこれをもとに実装する。
- 医療データを扱うため、本番DB接続情報・JWTシークレットは必ず環境変数で管理すること
- タブレット動作確認はiPadの実機（またはシミュレータ）で行うこと
