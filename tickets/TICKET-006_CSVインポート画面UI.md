# TICKET-006: CSVインポート画面UI

## 概要
ドクター・スタッフがCSVをアップロードしてデータを取り込む管理画面を実装する。

## 画面仕様

### インポート画面 `/import`
- 医院選択ドロップダウン
- インポート種別選択（患者情報 / 診療履歴 / 予約情報）
- ドラッグ＆ドロップ対応ファイルアップロードエリア
- アップロードボタン
- 処理中スピナー表示
- 結果表示エリア
  - 成功件数 / エラー件数
  - エラー一覧テーブル（行番号・エラー内容）
  - エラーCSVダウンロードボタン

### インポート履歴画面 `/import/history`
- import_logsの一覧表示
- 日時・ファイル名・種別・件数・ステータス
